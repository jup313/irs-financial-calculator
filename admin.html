📖 Backup & Restore Guide

JSON Backup — Recommended for routine backups. Includes all client cases, contacts, invoices, notes, email history and settings. Does not include user passwords (by design). Restore by uploading the JSON file in the Restore panel.


Raw .db File — Full SQLite database including user accounts and password hashes. Use for disaster recovery or full server migration. You can now restore it directly from the Restore panel above — the server automatically validates the file, saves a pre-restore backup, and hot-swaps the database without restarting.


Scheduled Backups — Set a recurring calendar reminder to download a JSON backup weekly or monthly. Store backups in a secure, separate location.


QNAP volume path: Container Station → Volumes → irs-db → Browse. The database file is at users.db inside the volume.
